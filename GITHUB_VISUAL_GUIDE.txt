════════════════════════════════════════════════════════════════════════════════
     HOW TO UPLOAD TO GITHUB - VISUAL STEP-BY-STEP GUIDE
                    For Ubuntu 22.04 Terminal
════════════════════════════════════════════════════════════════════════════════

═══════════════════════════════════════════════════════════════════════════════
 STEP 1: CONFIGURE GIT (Do this ONCE on your computer)
═══════════════════════════════════════════════════════════════════════════════

Open Terminal and type:

┌─────────────────────────────────────────────────────────────────────────────┐
│ $ git config --global user.name "Your Name"                                │
│ $ git config --global user.email "your.email@gmail.com"                    │
└─────────────────────────────────────────────────────────────────────────────┘

Replace "Your Name" and "your.email@gmail.com" with YOUR information!

EXAMPLE:
┌─────────────────────────────────────────────────────────────────────────────┐
│ $ git config --global user.name "Ahmed Ali"                                │
│ $ git config --global user.email "ahmed.ali@gmail.com"                     │
│ $ git config --global user.name                                             │
│   Ahmed Ali                                                                 │
│ $ git config --global user.email                                            │
│   ahmed.ali@gmail.com                                                       │
└─────────────────────────────────────────────────────────────────────────────┘

✓ If output shows your name/email → SUCCESS!

═══════════════════════════════════════════════════════════════════════════════
 STEP 2: CREATE PROJECT FOLDER
═══════════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────────┐
│ $ mkdir embedded-c-learning                                                │
│ $ cd embedded-c-learning                                                   │
│ $ pwd                                                                       │
│   /home/username/embedded-c-learning                                       │
└─────────────────────────────────────────────────────────────────────────────┘

✓ You're now inside your project folder

═══════════════════════════════════════════════════════════════════════════════
 STEP 4: CREATE .gitignore FILE
═══════════════════════════════════════════════════════════════════════════════

COPY AND PASTE THIS ENTIRE COMMAND (it's one big command):

┌─────────────────────────────────────────────────────────────────────────────┐
│ $ cat > .gitignore << 'EOF'                                                │
│ # Compiled programs (DON'T UPLOAD THESE)                                  │
│ 01_hello_embedded                                                           │
│ 02_pointers_arrays                                                          │
│ 03_structs_bitfields                                                        │
│ 04_practical_tasks                                                          │
│ 05_functions_modularity                                                     │
│                                                                              │
│ # Object files                                                              │
│ *.o                                                                         │
│ *.obj                                                                       │
│ *.exe                                                                       │
│ *.out                                                                       │
│ *.app                                                                       │
│                                                                              │
│ # IDE files                                                                 │
│ .vscode/                                                                    │
│ .idea/                                                                      │
│ *.swp                                                                       │
│                                                                              │
│ # System files                                                              │
│ .DS_Store                                                                   │
│ Thumbs.db                                                                   │
│ EOF                                                                         │
└─────────────────────────────────────────────────────────────────────────────┘

VERIFY:
┌─────────────────────────────────────────────────────────────────────────────┐
│ $ cat .gitignore                                                            │
│   (should show all the content above)                                       │
└─────────────────────────────────────────────────────────────────────────────┘

✓ .gitignore file created!

═══════════════════════════════════════════════════════════════════════════════
 STEP 7: CREATE GITHUB REPOSITORY (ON GITHUB WEBSITE)
═══════════════════════════════════════════════════════════════════════════════

BROWSER STEPS:

1. Go to: https://github.com
2. Log in (if not already logged in)
3. Click the "+" icon in top-right corner
4. Select "New repository"

SCREENSHOT (TEXT VERSION):
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                              │
│  GitHub > New Repository                                                   │
│                                                                              │
│  Repository name *                                                          │
│  ┌─────────────────────────────────────────────────────────────────────┐  │
│  │ embedded-c-learning                                                 │  │
│  └─────────────────────────────────────────────────────────────────────┘  │
│                                                                              │
│  Description (optional)                                                     │
│  ┌─────────────────────────────────────────────────────────────────────┐  │
│  │ C programming fundamentals for embedded systems and drone dev      │  │
│  └─────────────────────────────────────────────────────────────────────┘  │
│                                                                              │
│  Public ◉  (choose this)                                                   │
│  Private ○                                                                 │
│                                                                              │
│  ✓ Add a README file                                                       │
│  .gitignore template: [C ▼]                                                │
│  License: [No license ▼]                                                   │
│                                                                              │
│  [Create repository]                                                        │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘

5. Fill in details as shown above
6. Click "Create repository"

═══════════════════════════════════════════════════════════════════════════════
 STEP 9: CONNECT GITHUB TO YOUR LOCAL FOLDER
═══════════════════════════════════════════════════════════════════════════════

Back in Terminal:

┌─────────────────────────────────────────────────────────────────────────────┐
│ $ git remote add origin https://github.com/YOUR-USERNAME/embedded-c-learning.git
│                                                                              │
│ (Replace YOUR-USERNAME with YOUR actual username!)                         │
│                                                                              │
│ EXAMPLE:                                                                    │
│ $ git remote add origin https://github.com/ahmed123/embedded-c-learning.git
│                                                                              │
│ $ git remote -v                                                            │
│   origin  https://github.com/ahmed123/embedded-c-learning.git (fetch)    │
│   origin  https://github.com/ahmed123/embedded-c-learning.git (push)     │
└─────────────────────────────────────────────────────────────────────────────┘

✓ Connected!

═══════════════════════════════════════════════════════════════════════════════
 STEP 10: UPLOAD TO GITHUB (THE BIG MOMENT!)
═══════════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────────┐
│ $ git branch -M main                                                       │
│ $ git push -u origin main                                                  │
└─────────────────────────────────────────────────────────────────────────────┘

FIRST TIME - AUTHENTICATION:

GitHub might ask for password. You have options:

OPTION 1: GitHub Personal Access Token (RECOMMENDED)
┌─────────────────────────────────────────────────────────────────────────────┐
│ Username for 'https://github.com': your-username                           │
│ Password for 'https://your-username@github.com':                           │
│                                                                              │
│ Don't use your regular password!                                           │
│ Create a Personal Access Token:                                            │
│                                                                              │
│ 1. Go to: https://github.com/settings/tokens                              │
│ 2. Click "Generate new token" → "Generate new token (classic)"            │
│ 3. Name it: "Git Ubuntu"                                                  │
│ 4. Check ✓ "repo"                                                          │
│ 5. Click "Generate token"                                                 │
│ 6. COPY the token (shows once!)                                           │
│ 7. Paste it when asked for password                                       │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
