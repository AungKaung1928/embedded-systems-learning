════════════════════════════════════════════════════════════════════════════════
                      ⚡ SUPER QUICK SUMMARY (The Whole Process)
════════════════════════════════════════════════════════════════════════════════

1. Configure Git (one time):
   git config --global user.name "Your Name"
   git config --global user.email "your.email@gmail.com"

2. Create a folder:
   mkdir embedded-c-learning
   cd embedded-c-learning

3. Copy files:
   cp /mnt/user-data/outputs/*.c .
   cp /mnt/user-data/outputs/*.txt .
   cp /mnt/user-data/outputs/*.md .

4. Create GitHub repo:
   Go to https://github.com → New Repository → Create

5. Initialize Git:
   git init
   git add .
   git commit -m "Initial commit"

6. Connect to GitHub:
   git remote add origin https://github.com/YOUR-USERNAME/embedded-c-learning.git
   git branch -M main

7. Upload:
   git push -u origin main

8. Done! ✅

STEP 1: OPEN TERMINAL (2 minutes)

Windows: Install Git Bash or WSL
Mac: Terminal.app (Cmd+Space → Terminal)
Ubuntu: Already open or Ctrl+Alt+T

═══════════════════════════════════════════════════════════════════════════════

STEP 2: CONFIGURE GIT (2 minutes)

First time only:

$ git config --global user.name "Your Full Name"
$ git config --global user.email "your.email@gmail.com"

Example:
$ git config --global user.name "Ahmed Ali"
$ git config --global user.email "ahmed@gmail.com"

Verify it worked:
$ git config --global user.name
(should print your name)

═══════════════════════════════════════════════════════════════════════════════

STEP 3: CREATE PROJECT FOLDER (2 minutes)

$ mkdir embedded-c-learning

STEP 4: COPY ALL FILES (2 minutes)
