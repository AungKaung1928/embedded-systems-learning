## üìÅ Repository Structure

### **01_C_Programming/**
Foundation C programming for embedded systems with hands-on examples.
- Data types, pointers, memory management
- Bit manipulation and hardware register access
- Structures, unions, and efficient data organization
- Practical embedded patterns and real-time code

### **02_Electrical_Circuits/**
Essential electrical circuit theory for embedded engineers.
- Ohm's Law, Kirchhoff's Laws
- Passive components (R, L, C)
- AC/DC analysis
- Transistors and basic digital circuits

### **03_Embedded_Systems_Fundamentals/**
Core embedded systems concepts with code examples.
- Microcontroller architecture
- Memory types and organization
- GPIO, UART, SPI, I2C communication
- Interrupts, timers, and PWM
- DMA and power management

### **04_Control_Systems/**
Control theory fundamentals with C implementations.
- Open-loop vs Closed-loop systems
- PID controller design and tuning
- Transfer functions and stability
- Digital filters and state machines

### **05_Firmware_Development/**
Professional firmware development practices.
- Firmware architecture patterns
- RTOS concepts (FreeRTOS examples)
- Bootloader basics
- State machines and event-driven programming

### **06_MATLAB_Simulink/**
MATLAB/Simulink for control system design and simulation.
- MATLAB basics and control toolbox
- PID tuning and analysis
- Simulink modeling fundamentals
- Code generation for embedded targets
